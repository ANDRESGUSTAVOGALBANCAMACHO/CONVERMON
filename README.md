# CONVERMON - Conversor de Divisas

es una aplicación de consola desarrollada en Java que permite convertir entre varias divisas utilizando datos en tiempo real.

## Características

- Convierte entre diferentes divisas (USD, ARS, BRL, COP).
- Utiliza una API externa para obtener las tasas de conversión en tiempo real.
- Interfaz de consola interactiva.
- Requiere Java 11 o superior.
